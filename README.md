## Sobre o projeto

O **GeoAlert** é um site informativo que simula uma plataforma de alerta precoce para desastres naturais.

A ideia do projeto é mostrar como dados de risco poderiam ser organizados em uma interface simples, clara e acessível para apoiar moradores, comunidades vulneráveis e órgãos como a Defesa Civil.

O site apresenta:

- O conceito do GeoAlert
- Como a solução funciona
- Um painel simulado de alertas
- Cards com níveis de risco
- FAQ interativo
- Formulário de contato com validação
- Página de integrantes

---

## Objetivo

Criar um site front-end para apresentar o **GeoAlert** de forma:

- Clara
- Responsiva
- Acessível
- Organizada
- Coerente com o tema de prevenção de desastres naturais

## Como clonar e rodar

Para clonar o projeto na sua máquina, use o comando:

```bash
git clone https://github.com/manuelalramos/GeoAlert_Project.git
```

Depois, entre na pasta do projeto:

```bash
cd GeoAlert_Project
```

Como o projeto foi feito com HTML, CSS e JavaScript puro, é só abrir o arquivo `index.html` no navegador.

Para visualizar tudo certinho, precisa estar com internet, porque o projeto usa recursos externos como:

- Ícones do Font Awesome
- Fonte do Google Fonts
- Imagens externas do README
